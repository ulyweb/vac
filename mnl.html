This is the non-technical checklist for relatives. Gather everything first, then fill the forms one country at a time. Do not rush through forms because typo mistakes in passport number, birthdate, flight number, or arrival date can create airport delays.

PHStep 3Review
Verify

Review before submitting

Most airport problems come from typos or wrong travel direction.

+

Read aloud check

  1. Name matches passport.
  2. Passport number matches passport.
  3. Birthdate is correct.
  4. Arrival/departure direction is correct.
  5. Flight number and date are correct.
  6. Philippines address is usable.

After submit

  • Wait for the QR or registration proof.
  • Screenshot it immediately.
  • Save it to a shared folder.
  • Print one paper copy for seniors.

If there is no QR or you made a mistake

Do not panic. Reopen the official site and follow the portal’s correction/re-registration instructions. If unsure, ask airline staff at document check, but only after trying to fix it before leaving home.

Do not use fake links

The correct site is etravel.gov.ph. If a page asks for a suspicious payment or does not look like the official government system, stop.

Older traveler tip

Keep a printed sheet listing every traveler’s flight number and address. It prevents hunting through messages while filling the form.

Register

Register traveler and planned entry

Each traveler needs accurate passport and trip details.

+

Basic sequence

  1. Create/login to account.
  2. Register user information from passport.
  3. Add family members if managing multiple travelers.
  4. Register planned entry/return to Japan.

Senior-friendly note

Visit Japan Web can feel long because it separates user registration, trip registration, immigration, and customs. Finish one screen, screenshot, then continue. Keep the physical passport available because Japan may still require the passport itself.
